# Extensão

Anteriormente falamos que arquivos possuem um `nome.formato` esse `.formato` nada mais é que o identificador daquele aquivo, ele diz para o sistema de que forma ele deve ser usado.

Imagine que você precisa guardar um pequeno texto, um formato comum usado é o `.txt`, agora imagine que você deseja salvar uma música? O `.txt` não vai poder ser usado, pois o sistema entende que esse formato é usado para guardar texto e não músicas, para isso usamos outros como `.mp3`.

```bash
# texto
texto.txt

# audio
musica.mp3
```

Segue uma lista de tipos de arquivos:

`texto`
Arquivos de texto são compostos por caracteres legíveis por humanos.

```bash
texto.txt   # .txt: arquivo de texto simples.
texto.md    # .md: arquivo formatado em Markdown.
dados.csv   # .csv: valores separados por vírgulas.
config.json # .json: dados estruturados.
```

`audio`
Arquivos de áudio armazenam dados sonoros em diversos formatos de compressão e qualidade.

```bash
musica.mp3   # .mp3: áudio compactado, muito comum.
efeito.wav   # .wav: áudio de alta qualidade, sem compressão.
podcast.ogg  # .ogg: formato de áudio aberto e livre.
estudio.flac # .flac: áudio compactado sem perda de qualidade.
```

`script`
Scripts são arquivos de texto que contêm sequências de comandos para serem executados por um interpretador.

```bash
backup.sh    # .sh: script de shell (geralmente Bash).
processa.py  # .py: script escrito em linguagem Python.
app.js       # .js: script escrito em JavaScript.
```
